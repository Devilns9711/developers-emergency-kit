# Developer's Emergency Kit 🚀

The **Ultimate Toolkit Every Developer Needs**. Built completely from scratch with zero heavy frameworks, this local-first web application provides 8 essential mini-tools designed to make your daily development tasks faster and easier.

---

## 🛠️ Included Tools

1. **Password Generator** 🔑
   Generate highly secure passwords with customizable length and character sets. Includes a real-time strength meter based on cryptographic entropy.

2. **Color Picker** 🎨
   Pick, convert (HEX/RGB/HSL), and copy colors instantly. Also includes an analogous color palette generator and a customizable CSS gradient generator.

3. **JSON Formatter** 📝
   Paste ugly JSON and instantly beautify, minify, or validate it. Features syntax highlighting and detailed error reporting.

4. **Base64 Encoder / Decoder** 🔒
   Encode and decode plain text or entire files. Includes drag-and-drop support and automatic image previews for valid Base64 image strings.

5. **Lorem Ipsum Generator** 📖
   Generate placeholder text on the fly. Choose between words, sentences, or paragraphs with precise length controls.

6. **Character Counter** 🧮
   Get live, instant statistics on your text including character, word, sentence, paragraph, and space counts. Also calculates estimated reading and speaking times.

7. **QR Code Generator** 📱
   Instantly generate high-quality QR codes for URLs, text, emails, or phone numbers. Customize the foreground/background colors and download directly as a PNG.

8. **Timestamp Converter** ⏰
   Convert seamlessly between Unix timestamps and human-readable dates. Features a live clock and time zone converter.

---

## ✨ Features

- **Blazing Fast**: Pure Vanilla HTML, CSS, and JavaScript. Zero frameworks, zero bloat.
- **Privacy First**: Everything runs entirely in your local browser. No data is ever sent to a server.
- **Premium Design**: Beautiful, responsive layout with glassmorphism effects, smooth animations, and a modern aesthetic.
- **Dark Mode Default**: Built specifically with developers in mind, featuring a seamless Dark/Light theme toggle that persists across sessions.
